### Overview
Beacon is a streamlined Streamlit application that helps Minecraft enthusiasts discover and explore active game servers. With an intuitive interface, users can:

- Browse through paginated server lists
- Filter servers by player count
- View server details like name, IP, version, and tags
- Download server information as CSV

### Features
- Real-time server data scraping
- Interactive player count filtering
- Easy server list navigation
- CSV export functionality

### Requirements
- Python 3.8+
- Streamlit
- Requests
- BeautifulSoup
- Pandas

### Quick Start
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run main.py`

### Data Source
Server information sourced from [minecraft-mp.com](https://minecraft-mp.com)
